# Финальный проект OTUS. Универсальный генератор фич
Проект находится в состоянии разработки...

## Цель

Изучить возможность использования универсальных фич для:
* поиска изображений в БД
* детекции объектов в исходных изображениях
* сегментации объектов в исходных изображениях

## Задачи

1. Загрузить датасет `COCO-2017`
2. С помощью модели `dinov2` сгенерировать фич-векторы для всех изображений
3. С помощью модели `yolov8` сгенерировать массив детекций и сегметационных масок для всех изображений
4. Используя фичи из п.2. и детекцию-сегментацию из п.3. обучить кастомные модели
