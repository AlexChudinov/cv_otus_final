# Финальный проект OTUS. Универсальный генератор фич
В разработке...

## Цель

Изучить возможность использования универсальных фич для поиска изображений в БД по текстовому запросу

## План

1. ~~Загрузить датасет `COCO-2017`~~
2. ~~С помощью модели `dinov2` сгенерировать фич-векторы для всех изображений~~
3. С помощью текстового эмбедера построить эмбеддинги для `captions` в `COCO-2017`
4. Разработать|найти архитектуру, которая трансформирует фичи `dinov2` в эмбеддинги текстов

## Запуск
```bash
dpcker compose -f infra/docker-compose.yaml up --build
```
